- Usuário administrador
	O administrador do sistema deve ser capaz de operar sobre todas as entidades da aplicação. Assim como:
		1. Criar, ler e atualizar e deletar os dados de um Livro;
            1.1 - Criar Livro (cria uma entidade livro no banco de dados)
                O livro deve ter os seguintes campos: id, isbn, titutlo, subtitulo, lancamento, editora, quantidade de copias, autores, categoria
            1.2 - Ler Livro (busca uma entidade livro no banco de dados)
                Deve ser possível procurar um livro pelo seu id, isbn,  titulo, subtitulo, editora, autores e categoria.
            1.3 - Atualizar Livro (atualiza uma entidade livro no banco de dados)
                Deve ser possível atualizar as todas as informações de um livro no banco de dados, desde que essas informações não sejam de carater unico, como por exemplo o id e o isbn do livro.
            1.4 - Deletar Livro (deleta um determinado livro do banco de dados)
                Deve ser possível deletar um livro do banco de dados sem que haja mudanças nas entidades das quais ele depende.
		
        2. Criar, ler e atualizar e deletar os dados de um Autor;
            2.1 - Criar Autor (cria um autor no banco de dados)
                Os dados do autor guardados no banco de dados devem ser os seguintes:
                    id, Nome (Primeiro nome e sobrenome), data de nascimento, nacionalidade, contato, sexo.
            2.2 - Ler Autor (busca uma entidade autor do banco de dados)
                Deve ser possível resgatar todos os dados de um autor específico do banco de dados
            2.3 - Atualizar Autor (atualiza as inofrmações de uma entidade autor no banco de dados)
                Deve ser possível atualizar todos os dados de um autor (menos dados críticos)
            2.4 - Deletar Autor (deleta uma entidade autor do banco de dados)
                Deve ser possível deletar um determinado autor do banco de dados.
		
        3. Criar, ler e atualizar e deletar os dados de uma Categoria;
            3.1 - Criar Categoria
                O dados que devem ser guardados de uma categoria são as seguintes:
                    id, nome da categoria, descrição ou resumo.
            3.2 - Ler Categoria
            3.3 - Atualizar Categoria
            3.4 - Deletar Categoria
		
        4. Criar, ler e atualizar e deletar os dados de um Funcionario;
            4.1 - Criar Funcionario
            4.2 - Ler Funcionario
            4.3 - Atualizar Funcionario
            4.4 - Deletar Funcionario
		
        5. Criar, ler e atualizar e deletar os dados de um Professor;
            5.1 - Criar Professor
            5.2 - Ler Professor
            5.3 - Atualizar Professor
            5.4 - Deletar Professor
		
        6. Criar, ler e atualizar e deletar os dados de um Aluno;
            6.1 - Criar Aluno
            6.2 - Ler Aluno
            6.3 - Atualizar Aluno
            6.4 - Deletar Aluno
		
        7. Criar, ler e atualizar e deletar os dados de um Curso;
            7.1 - Criar Curso
            7.2 - Ler Curso
            7.3 - Atualizar Curso
            7.4 - Deletar Curso

        8. Criar, ler e atualizar e deletar os dados de um Curso;
            8.1 - Criar Curso
            8.2 - Ler Curso
            8.3 - Atualizar Curso
            8.4 - Deletar Curso
